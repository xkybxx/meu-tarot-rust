# 🔮 Oráculo de Tarot em Rust

Este é um projeto desenvolvido para explorar a linguagem **Rust**, focando em lógica de programação, manipulação de vetores e segurança de memória. O sistema simula uma tiragem de cartas de Tarot, fornecendo orientações baseadas no sistema de Arcanos.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Rust 🦀
* **Ambiente:** VS Code
* **Versionamento:** Git & GitHub

## 🛠️ Como rodar o projeto
Se você tiver o Rust instalado (Cargo), basta rodar:
```bash
cargo run
